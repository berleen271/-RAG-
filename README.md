## 二、研究目的
本研究旨在突破传统RAG系统在复杂文档理解场景下的核心瓶颈——单一文本模态处理能力与文档中图文混合、表格密集、布局多样的现实需求之间的结构性矛盾。

具体目标：
1. 构建面向复杂文档的多模态RAG系统架构，实现文本、表格、图表统一理解
2. 设计**分层索引**（页级→块级）与**双模态表示增强**方法，提升跨页与表格/图表检索精度
3. 构建专项评测集，系统评估多模态RAG相比纯文本RAG的性能增益
4. 实现**证据可追溯**的智能问答，答案自动关联页码、图号、表号
5. 交付可运行的Web演示系统与完整实验报告

## 三、研究意义
### 理论意义
1. 推动RAG从单模态向多模态演进，丰富多模态信息检索理论体系
2. 验证“页级粗筛+块级精排”分层检索与表格双模态表示的有效性
3. 探索检索粒度与内容理解的协同机制，为复杂文档理解提供方法参考

### 实践意义
1. 满足学术论文、财务报表、技术手册、法律文书等真实复杂文档处理需求
2. 提供可落地、可复现的多模态RAG工程方案
3. 降低表格/图表信息丢失率，提升问答可信度与可解释性

---

## 四、国内外研究现状
1. **多模态RAG（MRAG）** 已成为视觉富文档问答主流范式
2. **RAG-Anything**：一站式多模态抽取与检索系统
3. **VisDoMRAG**：多文档双模态问答，性能提升12–20%
4. **MDocAgent**：多智能体协同文档理解，准确率提升12.1%
5. 现有不足：检索粒度单一、分层机制薄弱、表格双模态表示未标准化

---

## 五、研究目标与技术指标
### 总体目标
构建面向图文混合复杂文档的多模态RAG系统，实现高精度、可溯源、可量化的智能问答。

### 核心指标
1. 跨页检索召回率（Recall@5）提升 ≥10%
2. 表格/图表问答准确率提升 ≥15%
3. 证据溯源覆盖率 ≥90%，引用准确率 ≥85%
4. 完整方案相比纯文本RAG准确率提升 ≥20%
5. 支持10页以上PDF，单轮问答响应≤30秒

---

## 六、研究内容与技术路线
### 主要研究内容
1. **复杂文档多模态解析**：抽取文本、表格、图表与版面信息
2. **分层索引构建**：页级粗筛 → 块级精排
3. **双模态增强**：表格=文本Markdown+图像；图表=描述+图像
4. **动态模态融合**：按问题类型自动调整文本/视觉检索权重
5. **答案生成与溯源**：大模型生成+事实校验+页码标注
6. **消融实验**：对比分层、双模态、动态权重的贡献

### 技术路线
PDF解析 → 多模态编码（文本+图像）→ 分层索引 → 双通道检索 → 重排序 → 生成答案 → 溯源校验

---

## 七、系统架构
1. **文档解析层**：PyMuPDF + OCR 提取图文表格
2. **多模态编码层**：Sentence-BERT文本向量 + CLIP视觉向量
3. **索引检索层**：页级索引 + 块级索引 + 跨模态检索
4. **生成溯源层**：大模型生成 + 证据引用 +  hallucination 校验
5. **展示层**：Streamlit Web界面 + 证据高亮可视化
